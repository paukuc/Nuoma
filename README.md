# Sprendžiamo uždavinio aprašymas

## Sistemos paskirtis

Projekto tikslas – sukurti patogią ir efektyvią platformą skirtą butų nuomai ir nuomai kur tiek nuomotojai tiek nuomininkai galėtų lengvai rasti vienas kitą susitarti dėl sąlygų ir atlikti visus reikiamus veiksmus elektroniniu būdu.

Veikimo principas – sistema susideda iš vartotojo sąsajos kurioje galima įkelti butų skelbimus, peržiūrėti prieinamus butus, užsisakyti peržiūras, ir administratoriaus panelės skirtos moderuoti skelbimus ir tvarkyti vartotojų duomenis.

## Funkciniai reikalavimai

- Svečias galės:
  - Naršyti butų skelbimus;
  - Užsiregistruoti ir prisijungti.

- Registruotas vartotojas galės:
  - Įkelti buto skelbimą;
  - Naršyti butų skelbimus;
  - Atsijungti;
  - Išsinuomuoti butą;
  - Atšaukti buto nuomą;

- Administratorius galės:
  - Moderuoti butų skelbimus (redaguoti, šalinti);
  - Moderuoti vartotojus (redaguoti, šalinti);
  - Įkelti buto skelbimą;
  - Naršyti butų skelbimus;
  - Atsijungti;
  - Išsinuomuoti butą;
  - Atšaukti buto nuomą;

## Sistemos architektūra

Sistemos sudedamosios dalys:
- Kliento pusė – naudojant React.js;
- Serverio pusė – naudojant .NET; Duomenų bazė – MySQL.
